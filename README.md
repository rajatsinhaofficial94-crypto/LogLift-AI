# 💪 LogLift

> **For those who know what they are doing.**

A sleek, offline-capable gym workout tracker built as a Progressive Web App (PWA). Log your lifts, track your progress, and review your history — all without an internet connection.

## 📱 Install on Android (Recommended)

LogLift is a **Progressive Web App** — it installs directly to your Android home screen and works like a native app, even offline.

### Steps:

1. Open **Google Chrome** on your Android phone
2. Visit: `https://69d669982b67921f5f03ca3b--splendorous-cassata-888411.netlify.app/`
3. Tap the **⋮ menu** (three dots) in the top-right corner
4. Tap **"Add to Home screen"**
5. Tap **"Add"** on the confirmation prompt
6. **Done!** LogLift will appear on your home screen as an app icon

> **Tip:** Once installed, the app works completely **offline**. Open it from your home screen and your Wi-Fi/Data won't affect it.

## ✨ Features

- 🏋️ **Active Workout Tracking** — Log exercises, sets, reps, weight and RIR in real-time
- ⏱️ **Workout Timer** — Built-in digital stopwatch with an analog clock visual
- 📅 **History View** — Calendar and list views of all your past sessions
- 📊 **Exercise Analytics** — Per-exercise volume, strength, and endurance trend charts
- 🏆 **My Lifts** — All-time personal records and exercise archive
- 🌙 **Dark Mode** — High contrast, sleek dark theme optimised for mobile
- 📴 **Offline First** — All data stored locally on your device, no account needed

## 🧑‍💻 Local Development

```bash
# Clone the repo
git clone https://github.com/rajatsinhaofficial94-crypto/LogLift.git
cd LogLift/app-preview

# Install dependencies
npm install

# Start dev server (accessible on local network)
npm run dev -- --host

# Build for production
npm run build
```

---

## 📦 Deploy

The `dist/` folder (generated by `npm run build`) can be deployed by dragging it onto [Netlify Drop](https://app.netlify.com/drop).
